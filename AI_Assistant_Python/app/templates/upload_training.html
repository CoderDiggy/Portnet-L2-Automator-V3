How It Works

Intelligent Column Detection: The system automatically detects incident and resolution columns based on:

  • Column Names: Looks for keywords like "incident", "problem", "resolution", "solution", etc.
  • Content Analysis: Analyzes cell content to identify incident descriptions and resolution steps
  • Smart Fallback: Uses the first text columns if specific patterns aren't found

Supported Formats: Excel files (.xlsx, .xls) with any column structure

Upload an Excel file with incident examples and their resolutions. The system will automatically detect the appropriate columns.
Tips for Best Results
  • Use descriptive column headers (e.g., "Incident Description", "Resolution Steps")
  • Include diverse incident types and detailed resolutions
  • Ensure incident descriptions are clear and specific
  • Resolution steps should be actionable and detailed
